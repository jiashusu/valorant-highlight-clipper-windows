Valorant Highlight Clipper

A local macOS web app for automatically clipping VALORANT highlights from kill-feed events. Vercel hosts this project page; video processing runs on your own Mac so it can access local recordings and ffmpeg.

View on GitHub Quick start

Why local?

VALORANT recordings are often several GB. The clipper needs local file access, macOS picker support, and long-running ffmpeg jobs. That makes a local app the right fit, while Vercel works well as the public project page.

Quick start

git clone https://github.com/jiashusu/valorant-highlight-clipper.git cd valorant-highlight-clipper python3 -m venv .venv source .venv/bin/activate python -m pip install -r requirements.txt bin/valorant_clipper_control.command start

Features

  • Drag video files or folders into the local web UI.
  • Export detected clips with estimated kill counts.
  • Strict filtering mode helps reduce teammate-kill false positives.
  • Start, stop, restart, and status controls for the local server.
