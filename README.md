# Valorant 高光剪辑 Windows 纯桌面版

这是 Windows 纯桌面应用版本，不打开浏览器，不启动本地网页服务。所有操作都在应用窗口里完成。

## 下载

请在本仓库的 Releases 页面下载：

```text
ValorantHighlightClipper.exe
```

exe 已内置识别资源、`ffmpeg.exe`、`ffprobe.exe` 和 `ffplay.exe`，下载后可直接双击运行。

## 本版修复

- 默认窗口加大到 1280x840。
- 扫描后自动选中第一条视频。
- 处理日志改为只读，只输出运行信息。
- Highlights 卡片区支持鼠标滚轮滚动。
- 后台导出、生成缩略图和生成低清预览时不再弹系统命令窗口。
- 默认本人击杀过滤更严格，减少队友击杀误剪。
- 默认导出改为高画质 H.264：`preset=slow`、`crf=14`，优先解决画质发糊。

## 功能

- 选择素材文件夹或单个视频
- 扫描视频列表
- 设置识别参数
- 后台剪辑并显示进度日志
- 导出区显示 3 列高光卡片
- 每张卡片显示低清缩略图、约 N 杀、片段长度、起止时间
- 点击卡片缩略图可在卡片内播放低清预览
- “高清播放”使用内置播放器播放真实 mp4
- 每张卡片支持打开目录和删除片段

## 使用方法

1. 双击 `ValorantHighlightClipper.exe`。
2. 在窗口里选择素材文件夹或单个视频。
3. 点击“扫描视频”，程序会自动选中列表里的第一个视频。
4. 按需要调整参数，然后点击“开始剪辑”。
5. 导出完成后，在右侧卡片墙里预览、播放或管理片段。

## 参数建议

默认参数已经偏向“少误剪”：

- 置信度：`0.93`
- 最短事件秒：`0.75`
- 严格过滤队友击杀：开启

如果漏掉自己的击杀，再小幅降低置信度或最短事件秒。
